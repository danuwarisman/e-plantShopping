# Paradise Nursery - Shopping Cart Application

A dynamic e-commerce web application for an online plant shop called Paradise Nursery. Users can browse plants, view details, add items to cart, and manage their shopping cart.

## Features
- Browse plants by categories
- Add/remove items from cart
- Update item quantities
- Real-time cart total calculation
- Responsive navigation bar
- Persistent cart state using Redux

## Technologies Used
- React.js
- Redux Toolkit
- React Router
- CSS3

## Installation
1. Clone the repository
2. Run `npm install`
3. Run `npm start`

## Project Structure
- `/src/components` - React components
- `/src/store` - Redux store and slices
- `/src/assets` - Images and static files
